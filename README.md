# fullstackopen-js
Full stack web app boilerplate using JS stack.

This project follows a monorepo-based architecture.

Tech Stack :

Frontend (Client) :
- React
- Redux
- Typescript
- Jest
- Bit

Backend (Server):
- Nodejs
- Express
- Typescript
- Graphql
- REST
- Swagger
- Mocha,Chai
- ESLint
- Yarn
- Babel

Database:
- Postgres
- Mongodb
- Redis

CI/CD:
- Kafka
- Docker
- Kubernetes
- AWS

Integrations
- Social Authentication
- RBAC
- Payment Integration (Stripe, Razorpay)
