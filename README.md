# fullstackopen-js
Full stack web app boilerplate using JS stack.

This project has a monorepo architecture managed using [Lerna](https://github.com/lerna/lerna)

Tech Stack :

Frontend (Client) :
- React
- Redux
- Typescript
- Jest

Backend (Server):
- Nodejs
- Express
- Typescript
- Graphql
- REST
- Swagger
- Mocha,Chai
- ESLint
- Yarn
- Babel

Database:
- Postgres
- Mongodb
- Redis

CI/CD:
- Kafka
- Docker
- Kubernetes
- AWS

Architechture:
- Microservices
- Multi-tenant

Integrations
- Social Authentication
- Payment Integration
